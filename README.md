# 영어 단어 카드 학습 앱

초등학생을 위한 영어 단어 카드 학습 페이지입니다. 교사가 구글 시트로 단어를 관리하고, 학생은 카드 학습과 4지선다 게임으로 공부합니다.

## ✨ 주요 기능

- 📖 **카드 학습 모드**: 단어 카드를 뒤집으며 뜻 확인, 자동 발음 재생
- 🎮 **4지선다 게임**: 틀린 단어는 라운드 내 다시 등장 (B안)
- 👩‍🏫 **선생님 모드**: 레슨/단어 직접 편집, JSON 내보내기/가져오기
- 🌐 **구글 시트 연동**: 시트 URL 하나로 전체 단어 세트 관리
- 🔗 **공유 링크**: `?sheet=...` 파라미터로 학생에게 단어 세트 배포

## 🛠 기술 스택

- 순수 HTML + CSS + JavaScript (프레임워크 없음)
- Web Speech API (영어 발음 재생)
- localStorage (레슨 데이터 캐시)
- Google Sheets 게시(Publish to Web) CSV (원격 데이터 소스)
- 주아체(Jua) 웹폰트

## 🚀 사용 방법

### 빠른 시작

```bash
open index.html
```

브라우저에서 바로 실행됩니다. 별도 서버 불필요.

### 구글 시트로 단어 관리

1. 구글 시트 생성 후 아래 형식으로 입력:

   | Lesson | Emoji | English | Korean |
   |---|---|---|---|
   | 1. Hello | 👋 | hello | 안녕하세요 |
   | 1. Hello | 👋 | name | 이름 |

2. **파일 → 공유 → 웹에 게시** → 형식 "쉼표로 구분된 값(.csv)" → 게시
3. 앱의 `선생님 모드` → 생성된 URL 붙여넣기 → `🔄 불러오기`
4. 생성된 `?sheet=...` 공유 링크를 학생에게 전달
