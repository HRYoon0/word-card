🌐 구글 시트로 단어 관리

  1. 위 버튼을 눌러 샘플 CSV 파일을 다운로드하세요.
  2. sheets.new에 접속해 빈 구글 시트를 만듭니다.
  3. 구글 시트 상단 메뉴: 파일 → 가져오기 → 업로드 → 방금 받은 CSV 선택
    → 가져오기 위치: "현재 시트 교체" → 데이터 가져오기
  4. 시트에서 단어를 자유롭게 수정/추가하세요.
    같은 Lesson 이름을 쓰면 자동으로 한 단원으로 묶입니다.
  5. 상단 메뉴: 파일 → 공유 → 웹에 게시
  6. 형식 쉼표로 구분된 값(.csv) 선택 → 게시 버튼 클릭
    → 경고창 나오면 확인
  7. 생성된 URL을 복사해 아래 입력창에 붙여넣고 🔄 불러오기 클릭

💡 기존 단어는 모두 시트 내용으로 교체돼요. 필요하면 먼저 아래 "📤 내보내기"로 백업하세요.
